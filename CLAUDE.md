# Ready Apps

Repository for Open Mercato example applications and their App Specs.

## Structure

- `app-specs/` — Business analysis zone. App Spec documents, domain knowledge, workflow analysis. Use Mat and Piotr skills here.
- `apps/` — Implementation zone. Actual application code that consumes Open Mercato packages.

## Working in this repo

- When working in `app-specs/`, do NOT load OM monorepo context. Mat and Piotr skills provide the methodology. Piotr fetches OM capabilities on-demand.
- When working in `apps/<app>/`, follow that app's CLAUDE.md for full implementation context.
